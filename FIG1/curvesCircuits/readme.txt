RunLtspice.py will run all simulations and will create the tuning curves CSV files.

Please make sure the LTspice simulator is installed on the computer.

To use RunLtspice.py you first need to install ltspice python lib by running :
pip install ltspice

In the script,
Please change LTSPICE_PATH to XVIIx64.exe location.
Please change CIRCUIT_FOLDER to the folder that contains: 

OZ_8_bounded_curves.asc
OZ_8_pure_curves.asc
OZ_8_uniform_curves.asc


Now the script can be run.
