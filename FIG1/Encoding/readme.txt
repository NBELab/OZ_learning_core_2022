RunLtspiceEncoding.py will run all simulations and will create the event graphs as .png file .

Please make sure the LTspice simulator is installed on the computer.

To use RunLtspiceEncoding.py  you first need to install ltspice python lib by running :
pip install ltspice

In the script,
Please change LTSPICE_PATH to XVIIx64.exe location.
Please change CIRCUIT_FOLDER to the folder that contains the circuit, folder path with glob pattern is supported as well.

Now the script can be run.
