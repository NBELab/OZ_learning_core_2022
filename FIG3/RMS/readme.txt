RunLtspiceRMS.py will run all simulations and will create the neurons RMS graphs as a .csv file.

Please make sure the LTspice simulator is installed on the computer.

To use RunLtspiceRMS.py  you first need to install ltspice python lib by running :
pip install ltspice

In the script,
Please change LTSPICE_PATH to XVIIx64.exe location.

Please change RESULTS_ROOT_FOLDER to FIG3\RMS.


Now the script can be run.
