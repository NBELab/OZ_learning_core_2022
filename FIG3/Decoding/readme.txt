RunLtspiceDecoding.py will run all simulations and will create the neurons decoding graphs as .csv file .

Please make sure the LTspice simulator is installed on the computer.

To use RunLtspiceDecoding.py  you first need to install ltspice python lib by running :
pip install ltspice

In the script,
Please change LTSPICE_PATH to XVIIx64.exe location.

Please change CIRCUIT_FOLDER to \FIG3\B-D full path.


Now the script can be run.
